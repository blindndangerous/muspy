About Muspy

Muspy tracks album and release information from MusicBrainz so listeners can follow artists and find upcoming releases.

How it works

Signed-in users follow artists. Muspy syncs public release data, lists visible release events, and can deliver notifications through email, RSS, or iCal feeds.

Source code

Muspy is free and open source software released under the GNU AGPL.
