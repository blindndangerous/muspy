Frequently Asked Questions

Where does release data come from?

Release data comes from MusicBrainz. Missing or incorrect artist and release data should be fixed in MusicBrainz first.

Do I need an account?

Public release pages and public API endpoints do not require an account. Following artists, imports, feeds, and notifications require signing in.

Why are some releases missing?

Only visible releases known to the current Muspy database are listed. Data freshness depends on sync jobs and upstream availability.
